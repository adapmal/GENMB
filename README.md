# GENMB 1.0.0-alpha.1

Primeira versão Foundation baseada no motor 1.2 Beta.

## Instalação
1. Extraia em uma pasta nova.
2. Execute `install.bat` uma vez.
3. Execute `run.bat`.
4. Abra `http://127.0.0.1:5050`.

Os dados persistentes ficam em `Documentos\GENMB\Data`. Na primeira execução, dados existentes em `Documentos\News Reconstruction Studio\Data` são copiados sem apagar a origem.

## Incluído nesta alpha
- Nome GENMB e nova pasta persistente.
- Migração não destrutiva dos dados antigos.
- Remoção do controle Fidelidade.
- Pesos tipográficos em escala 1–9.
- Escala de logo até 300%.
- Detecção de fontes do veículo.
- Renderização isolada da notícia selecionada no Preview.
- Gerenciamento de armazenamento e limpeza seletiva.
